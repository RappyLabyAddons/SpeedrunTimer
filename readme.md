### Installation
1. Press `Win` + `R`
2. Paste this into the window that popped up: `%appdata%/.minecraft/LabyMod-neo/addons` and press enter
3. It should open your Labymod addon directory; Paste the [SpeedrunTimer.jar](https://github.com/RappyLabyAddons/SpeedrunTimer/releases/latest/download/SpeedrunTimer.jar) in there.
4. Launch your Labymod client.
